# Personal Portfolio

The source for my personal portfolio site.

## About

Single-page portfolio built from scratch as part of the [Next Chapter](https://www.nextchapterproject.org/) AI Augmented Builder Bootcamp. Vanilla HTML, CSS, and JavaScript. No frameworks, no build step, no dependencies.

## Stack

- HTML5
- CSS3 (custom properties, grid, flexbox)
- Vanilla JavaScript (Intersection Observer for active nav highlighting)
- GitHub Pages for hosting

## Structure

```
.
├── index.html       Single-page markup
├── style.css        Design tokens and component styles
├── script.js        Active nav highlighting
└── images/          Profile pic and project screenshots
```

## Local development

No build step. Clone and open `index.html` in a browser:

```bash
git clone https://github.com/philay3/Personal-Portfolio.git
cd Personal-Portfolio
open index.html
```
