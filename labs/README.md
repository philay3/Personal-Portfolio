# Next Chapter Labs

A living journal of labs and exercises from the [Next Chapter](https://www.nextchapterproject.org/) AI Augmented Builder Bootcamp.

## About

Single-page app organized by week, day, and lab. Tree explorer in the sidebar, full lab writeup in the main pane. Built vanilla: no frameworks, no build step.

## Stack

- HTML5
- CSS3
- Vanilla JavaScript
- GitHub Pages

## Structure

```
.
├── index.html      Markup + tree explorer shell
├── style.css       Design tokens and component styles
├── script.js       Tree rendering and click handlers
├── labs.js         13 weeks of lab data
└── images/         Lab screenshots
```
