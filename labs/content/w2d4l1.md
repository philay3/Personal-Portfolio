### Notes

Worked through the three given functions (`greet`, `addNumbers`, `isEven`) in the DevTools console, then wrote three of my own: `square`, `shout`, and `isLong`.

### Code

```javascript
function square(n) {
  return n * n;
}
square(5);    // 25
square(12);   // 144

function shout(message) {
  return message.toUpperCase();
}
shout("hello");          // "HELLO"
shout("good morning");   // "GOOD MORNING"

function isLong(text) {
  return text.length > 10;
}
isLong("hi");                       // false
isLong("this is definitely long");  // true
```

### What I learned

A function takes inputs (parameters), does work, and returns a value. No `console.log` inside unless you want a side effect; `return` hands the value back to the caller. `.toUpperCase()` is a string method, which is just a function that lives on a value. And when an expression like `text.length > 10` already evaluates to a boolean, return it directly. No need for `if (...) return true; else return false;`.
