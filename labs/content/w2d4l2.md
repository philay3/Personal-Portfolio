### Notes

Started with the for...of loop over five names, then modified it three different ways to practice the patterns. Plus one more loop over my own array of weekly lessons.

**Modification 1: Classic for loop with index**

The classic for loop is the right tool when you need the index. `i + 1` because positions read 1-based to humans but arrays are 0-based.

```javascript
const names = ["Alex", "Sam", "Jordan", "Riley", "Casey"];

for (let i = 0; i < names.length; i++) {
  console.log(`${i + 1}. Hello, ${names[i]}!`);
}
```

Output:

```
1. Hello, Alex!
2. Hello, Sam!
3. Hello, Jordan!
4. Hello, Riley!
5. Hello, Casey!
```

**Modification 2: Filter inside for...of**

Only greet names longer than 4 characters. Alex (4) and Sam (3) get skipped; Jordan, Riley, and Casey get greetings.

```javascript
for (const name of names) {
  if (name.length > 4) {
    console.log(`Hello, ${name}!`);
  }
}
```

**Modification 3: Build a new array of greetings**

This is the data-shaping pattern that shows up everywhere: walk one array, build a new array with transformed values. `.map()` does this in one line, but writing it out as for...of first makes what `.map` actually does click.

```javascript
const greetings = [];
for (const name of names) {
  greetings.push(`Hello, ${name}!`);
}
console.log(greetings);
```

### What I learned

Loops are how you apply the same logic to every item in an array. for...of when you only need each value, classic for when you need the index, filtering with an if inside the loop, transforming by pushing to a new array. These four patterns cover most array work. `.map`, `.filter`, and `.reduce` are syntactic sugar on top of these patterns, useful once the patterns themselves are second nature.
